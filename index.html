Welcome to Git/GitHub Tutorials

Git is a version control as well code repository to manager our projects locally as well remotely

Please follow the below steps

  1. Download git from their website and install it
  2. Check about the installation is success or not? go to command prompt and type git --version and it will show the concern version of git and it's details. if not, right click your mycomputer and properties and environmental - path (here add the git installated location path like c:programeFiles/git/git.cmd)
  3. Open your project folder and select the path on explorer address bar and type cmd. cmd will open with the project folder location and type git init (git will initialize within your project folder)
    4. Once git initialize start to create staging. your developed your project upto certain point and you decided to create a stage that point to proceed further. cmd - git - add index.html
  4. check the status cmd- git status
  5. Commit the stage. cmd -git commit -m "your message here"
  6. again check the status cmd- git status

Create Branches for colleague or you (clone the master branch)

Repeat the above point nos 4-6 on the welcome to github tutorials section

We can use github desktop ui tool to do these stuffs (if you feel discomfort with command prompt)
